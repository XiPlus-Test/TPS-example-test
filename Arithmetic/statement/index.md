# 四則運算

數字 $x$，操作 $+a$，結果 $x+a$。  
數字 $X$，操作 $+A$，結果 $X+A$。  

編號 $1, 2, \cdots, n$  
編號 $1, 2, \cdots, N$  

第一行包含兩個整數 $n$、$m$。  
第一行包含兩個整數 $N$、$M$。  

接下來有 $q$ 行。  
接下來有 $Q$ 行。  

第二行包含 $N$ 個數字 $a_1, a_2, \cdots, a_N$。  
第二行包含 $N$ 個數字 $A_1, A_2, \cdots, A_N$。  
第二行包含 $n$ 個數字 $a_1, a_2, \cdots, a_n$。  
第二行包含 $n$ 個數字 $A_1, A_2, \cdots, A_n$。  

\clearpage

 - $1 \leq n \leq 10^4$
 - $1 \leq N \leq 10^4$
 - $1 \leq m \leq 10^4$
 - $1 \leq M \leq 10^4$
 - $1 \leq k \leq 10^4$
 - $1 \leq K \leq 10^4$
 - $1 \leq q \leq 10^4$
 - $1 \leq Q \leq 10^4$
